addEventListener is a universal method, because it allows to attache an event handler to the specified element. It doesn't overwrite existing event handlers. You can add as many event handlers to one element as you want. When using the addEventListener() method, the JavaScript is separated from the HTML markup, for better readability and allows you to add event listeners even when you do not control the HTML markup.
