11. Designing, Defining, and Triggering CSS3 Transitions without Custom Libraries

Transition is a tool, which makes the element smoothly and gradually change from one state to another. Very often it used together with transform (using which you can rotate, move, skew, and scale elements). However without a transition, an element being transformed would change abruptly from one state to another. Let's consider basic forms of transitions.
